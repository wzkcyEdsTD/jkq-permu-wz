# egg-react-mobx-antd-easywebpack

基于 egg-easywebpack 服务端渲染后台管理系统。

## how to use

_git clone 项目地址_<br/>
_yarn install 安装_<br/>
_yarn run dev 本地调试_<br/>
_yarm run build 生产打包_
